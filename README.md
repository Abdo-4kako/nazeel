## 📌 About The Project
**NAZEEL** is an ultra-luxury, high-performance web application designed for the top 1% of global travelers. Built with a focus on "Editorial Minimalist" design and cinematic user experience, it serves as a sovereign registry for booking the world's most exclusive properties. 

This project demonstrates advanced UI/UX engineering, real-time client-side filtering, and a strict clean-code architecture.

## ✨ Key Features
* **Cinematic UI/UX:** Custom `slowPan` and `slowZoom` CSS animations that create an immersive, breathing interface.
* **Editorial Design:** "Old Money" aesthetic utilizing precise typography tracking, whitespace, and high-contrast monochromatic palettes with muted gold accents.
* **Advanced Filtering Engine:** Real-time, multi-parameter search (Destination, Property Type, Price Range, and Signature Inclusions) built with React `useMemo` for absolute performance.
* **The Global Vault:** A comprehensive, heavily structured mock database featuring 16 ultra-luxury global sanctuaries across 12 countries.
* **Fully Responsive:** Flawless execution across mobile, tablet, and ultra-wide desktop displays without relying on traditional navigation bars, maximizing the visual real estate.

## 🛠️ Tech Stack

* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **Language:** TypeScript
* **Icons:** [Lucide React](https://lucide.dev/)
